This directory contains screenshots and images used in the main README.md file.

1. **order-dashboard.png** - Order Dashboard with Platform Events

5. **device-inventory-lwc.png** - Device Inventory Lightning Web Component

## Screenshot Guidelines

### Recommended Size
- Width: 1200-1600px
- Format: PNG (preferred) or JPG
- Quality: High resolution for clarity

### What to Capture

**Order Dashboard:**
- Show real-time order list
- Include status indicators
- Highlight Platform Events notification

**REST API (Postman):**
- Show GET/POST/PUT/DELETE endpoints
- Include request headers (Authorization)
- Show successful response JSON

**Platform Events Monitor:**
- Setup → Platform Events → OrderStatusChangeEvent__e
- Show event definition with fields
- Include subscriber view if possible

**Test Coverage Report:**
- Developer Console → Test → Run Tests
- Show overall coverage percentage (85%+)
- Include list of test classes

**Device Inventory LWC:**
- Show lightning component in action
- Display device list with data
- Show interactive features (search, filter, actions)
